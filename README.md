## Структура файлов

- `index.js` — основной файл бота
- `database.js` — работа с SQLite3
- `senators.js` — локальный состав сената и Discord-резолвинг
- `.env` — конфигурация

## Состав Сената

Состав хранится локально в SQLite и управляется прямо в Discord.

- `/replace_senator` — добавить сенатора, заменить его или деактивировать место
- `/refresh_senate` — синхронизировать локальный список с пользователями, у которых есть роль сенатора
- `setup_senate` — публикует главное сообщение с актуальным составом

## Команды
Многие команды не используются в повседневной деятельности, а подвязаны на кнопки!

| Команда | Описание |
|---------|----------|
| `/send` | Подать законопроект |
| `/create_meeting` | Создать заседание (председатель) |
| `/setup_senate` | Создать главное сообщение Сената |
| `/setup_bills_forum` | Настроить форум законопроектов |
| `/replace_senator` | Добавить, заменить или деактивировать сенатора |
| `/refresh_senate` | Синхронизировать состав из Discord |
| `/set` | Установить количество сенаторов - не использовать |
| `/help` | Справка |

## Новые возможности заседаний
